# Project Overview

This project implements a galaxy merger pair detection pipeline using JWST photometric catalogs.
It includes:
* Catalog merging
* Quality filtering (SNR and flag-based)
* Color-based classification
* Companion search in projected separation
* Redshift binning

Field Example:

CEERS field is used as a demonstration. The same pipeline was applied to 7 JWST deep fields.

Skills Demonstrated:

* Large-scale data filtering
* Astronomical coordinate handling
* Cosmological distance calculations
* Algorithmic neighbor search
* Feature engineering
* Reproducible scientific pipeline
  
Technologies:

* Python
* NumPy
* Astropy
* SciPy
